# Node.js란?

- Node.js는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임
- JavaScript라는 프로그래밍 언어가 동작하는 컴퓨터 환경

# npm이란?

- NPM(Node Package Manager)은 전 세계의 개발자들이 만든 다양한 기능(패키지,모듈)들을 관리

## 개발용 의존성 패키지 설치

- (-D, --save-dev)
- $ npm install -D
- 개발할 때만 사용

## 일반 의존성 설치

- $ npm install
- 실제 웹브라우저에서 동작할 수도 있음.
